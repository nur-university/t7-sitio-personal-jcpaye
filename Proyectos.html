Juego de Trivia

Este proyecto fue realizado durante mi segundo semestre en la carrera, mi persona Jose Carlos Paye Botero
junto a mi compañero Sebastian Gonzales hemos trabajado en este proyecto para la materia de Programación II.

El proyecto consistio en un juego de trivia, el cual cargaba un archivo TXT con las categorias, preguntas
y respuestas del juego de trivia, en la ventana inicial se mostraban las categorias disponibles de juego,
una vez seleccionada la categoria deseada, se creaba una nueva ventana donde comenzaban a mostrarse las
preguntas acorde a la categoria, en esta ventana se coloco un Timer para el cual el usuario disponia de 10
segundos para seleccionar la respuesta correcta a la pregunta, para cada pregunta se notificaba si la respuesta
era correcta o incorrecta, sumado a esto se otorgaba al usuario un boton para poder pasar a la siguiente pregunta.
Una vez se respondian 10 preguntas se mostraba un recuadro en la pantalla indicando el numero de aciertos sobre el total de preguntas.

Nota: El archivo TXT estaba estructurado mediante Espacios, Saltos de linea y Caracteres especiales para poder separar
las categorias, preguntas y respuestas. Para este proyecto solo se utulizo el Lenguaje Java mediante el compilador Intellij IDEA.

Registro/Login de Estudiantes

Este pryecto se realizo durante mi cuarto semestre en la carrera, fue realizado por mi persona Jose Carlos Paye Botero
junto a mi compañero Renzo Viera, para la materia de Base de Datos II.

El proyecto a rasgos generales se dividia en 2 partes, una interfaz grafica y una base de datos, por otro lado y de forma
mas detallada, el proyecto poseia 4 partes fundamentales: Ventana de Bienvenida, un Login, un Formulario y un Area Personal.

La ventana de Bienvenida de un mensaje y dos botones disponibles para el usuario, el primero de ellos era el boton
Login, y el segundo el boton Registrarse. El boton de Login abria una nueva ventana la cual poseia 2 campos para
rellenar, registro y contraseña. El boton Registrarse abria una ventana nueva la cual poseia un Formulario con diferentes
campos y al final un boton finalizar.

La ventana con el Formulario deberia ser la primera en ser accedida en caso de que el usuario no haya hecho un registro
previamente, el Formulario tenia campos para: Nombres, Apellidos, Correo Electronico, Contraseña, Telefono, Ciudad, Carrera, Semestre,
Fecha de Nacimiento y Genero. Cuando se oprimia el boton finalizar se mostraba en pantalla un mensaje con el registro
del usuario, este era generado mediante una funcion desarrollada en la Base de Datos, el registro era una combinacion
de las inciales de Nombres y Apellidos mas los numeros de la Fecha de Nacimiento. Luego de dar OK al mensaje,
automaticamente se abria la ventana de Login.

La ventana de Area Personal se mostrara luego de haber iniciado sesion en el Login y de haber oprimido en el boton
de Iniciar Sesion. El Area Personal contiene los datos del usuario ingresados en el Formulario. En esta ventana el usario
tambien podra modificar sus datos asi como elmininar su perfil, de forma adicional el usuario tendra una lista de materias
disponibles para inscribir, las materias estan restringidas por semestre.

Nota: Para el proyecto se utilizo una base de datos usando el sistema de PostgreSQL y Dveaver para gestionar la base de
datos, la interfaz grafica se la diseño usando Java en el compilador Intellij IDEA.
